# MERN & MSA 과제 - Server

## 💻개요

Express.js를 이용해 만든 서버입니다. 



## 📁프로젝트 시작

### yarn 설치

```bash
$ npm install -g yarn
```

### dependencies 설치

```bash
$ yarn
```

### package 설치

```bash
$ yarn add [package name]
```

### 로컬 테스트

```bash
$ cd server
$ node app
# 로컬 경로 | http://localhost:5000 
```
